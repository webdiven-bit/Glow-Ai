# TikVibe Creator Pro Backend API

This is the backend API for TikVibe Creator Pro - an advanced video editing platform with AI capabilities.

## API Endpoints

- `GET /health` - Health check
- `GET /api/test` - Test endpoint
- `POST /api/ai/velocity-edit` - Create velocity edit
- `POST /api/ai/beat-montage` - Create beat-synced montage
- `POST /api/ai/aesthetic-edit` - Create aesthetic edit
- `POST /api/ai/smooth-transitions` - Create smooth transition edit
- `POST /api/ai/vlog-edit` - Create vlog-style edit
- `POST /api/ai/imitate-style` - Imitate style from inspiration video
